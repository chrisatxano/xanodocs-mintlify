---
title: "Array"
---

An array is just another name for a list of items. These items can be anything from text, to integers, to full objects and complex data structures.

An array is always contained inside of square brackets: **\[ ]**

```
[
    1,
    2,
    3
]
```

```javascript
[
    "Hello",
    "World"
]
```
